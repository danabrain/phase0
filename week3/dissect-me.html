Git is a version control software that lives locally on your computer. Git allows you to save different versions of work as you go. On the other hand, Github lives remotely on the Internet so that programmers could share their code remotely and collaborate on projects at the same time.

Version control software allows you to save different versions of you work and lets you create and work on different branches that allow you to switch back and forth out of interchangeably. Github lives on the Internet and could help programmers to collaborate remotely.

Developers use version control to keep track of changes in their code. It becomes more and more important when you are working on the same files with collaborators. Instead of sending a file back and forth to work on it, developers can all work on the same files at the same time.
